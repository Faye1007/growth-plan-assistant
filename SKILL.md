## 核心概念

### 日程 vs 任务 vs 习惯 vs 灵感

- **日程**：固定必做的周期性事项，如每天早起、每周运动
- **任务**：一次性事项，如改简历、投简历
- **习惯**：需要每天打卡的习惯，如阅读、喝水
- **灵感**：想做但当下不能安排的事情，如学习某技能、去某地旅行

## 脚本使用

### 日程管理

```bash
# 创建日程（必要字段：日程名称、时间、重复规则、标签）
./scripts/create_schedule.sh "日程名称" "时间" "重复规则" "标签"
# 示例：./scripts/create_schedule.sh "吃异维A" "20:00" "每天" "健康"
# 重复规则：每天/每周一/每周二/.../每周日/每周三、周日
# 标签：健康/学习/工作/生活

# 修改日程
./scripts/update_schedule.sh "日程名称" "新时间" "新规则"

# 删除日程
./scripts/delete_schedule.sh "日程名称"
```

### 任务管理

```bash
# 创建任务（必要字段：任务名称、截止日期、分类）
./scripts/create_task.sh "任务名称" "截止日期" "分类" ["备注"]
# 示例：./scripts/create_task.sh "投简历5个" "2026-04-20" "工作"
# 示例（带备注）：./scripts/create_task.sh "看看书" "2026-04-20" "学习" "阅读运营书籍"
# 分类：学习/工作/生活

# 修改任务
./scripts/update_task.sh "任务名称" --status "已完成"

# 完成任务
./scripts/complete_task.sh "任务名称" ["复盘内容"]

# 删除任务
./scripts/delete_task.sh "任务名称"
```

### 习惯打卡

```bash
# 创建习惯（必要字段：习惯名称、标签）
./scripts/create_habit.sh "习惯名称" "标签"
# 示例：./scripts/create_habit.sh "快走10000步" "健康"
# 标签：健康/学习/工作/生活/爱好

# 打卡（可选复盘内容）
./scripts/checkin_habit.sh "习惯名称" ["复盘内容"]

# 打卡规则：
# - 简单打卡（无复盘）：只更新【习惯】表的最后打卡日期、连续天数、累计天数
# - 带复盘打卡：更新【习惯】表 + 写入【打卡记录】表
# - 相关表格：
#   - 习惯表ID：YOUR_HABIT_TABLE_ID
#   - 打卡记录表ID：YOUR_CHECKIN_TABLE_ID

# 修改习惯
./scripts/update_habit.sh "习惯名称" --target "新目标"

# 删除习惯
./scripts/delete_habit.sh "习惯名称"
```

### 灵感记录

```bash
# 记录灵感（必要字段：灵感内容）
./scripts/record_idea.sh "灵感内容"
```

### 查询

```bash
# 今日汇总（自动同步飞书数据）
./scripts/query_today.sh

# 每日清单（自动同步飞书数据）
./scripts/query_daily_list.sh

# 周数据
./scripts/query_week.sh

# 月数据
./scripts/query_month.sh
```

### 数据同步

```bash
# 从飞书日历/任务同步到多维表格
# 时间范围：前后一周
# 同步方向：飞书 → 多维表格
./scripts/sync_from_feishu.sh
```

**注意事项**：
- 同步以飞书任务ID为唯一标识，同名任务（不同截止日期）不会互相覆盖
- 同步只更新状态和日期，不会覆盖多维表格里已有的数据
- 同步时从飞书清单获取分类（学习/工作/生活），新增任务写入分类
- **重要**：多维表格字段索引（ID=0, 日期=1, 时间=2, 复盘=3, 飞书任务ID=4, 标题=5, 状态=6, 分类=7），修改表格结构后需同步更新脚本

### 关键配置项

| 配置项 | 占位符 | 说明 |
|--------|--------|------|
| Base Token | `YOUR_BASE_TOKEN` | 多维表格的访问令牌 |

| 打卡记录表 ID | `YOUR_CHECKIN_TABLE_ID` | 打卡记录表 |

| 工作清单 ID | `YOUR_WORK_LIST_ID` | 飞书任务-工作清单 UUID |
| 生活清单 ID | `YOUR_LIFE_LIST_ID` | 飞书任务-生活清单 UUID |
