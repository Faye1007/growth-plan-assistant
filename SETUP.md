# 成长计划助手 - 配置指南

## 1. 创建飞书多维表格

### 1.1 创建多维表格

1. 打开 [飞书](https://feishu.cn)，点击左上角「新建」→「多维表格」
2. 将表格命名为「我的成长计划」或你喜欢的名字
3. 创建以下数据表：

#### 表1：固定日程表

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 日程名称 | 文本 | 日程内容 |
| 开始时间 | 日期时间 | 日程开始时间 |
| 重复规则 | 单选 | 每天/每周一/每周二... |
| 状态 | 单选 | 进行中/已暂停/已结束 |
| 飞书日历事件ID | 文本 | 同步到飞书日历的事件ID |

#### 表2：任务表

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 日期 | 日期 | 任务日期 YYYY-MM-DD |
| 标题 | 文本 | 任务名称 |
| 具体时间 | 日期时间 | 任务具体时间 |
| 分类 | 单选 | 学习/工作/生活 |
| 状态 | 单选 | 未开始/进行中/已完成 |
| 复盘 | 文本 | 任务复盘内容 |
| 飞书任务ID | 文本 | 同步到飞书任务的任务ID |

#### 表3：习惯表

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 习惯名称 | 文本 | 习惯内容 |
| 标签 | 单选 | 健康/学习/工作/生活/爱好 |
| 累计天数 | 数字 | 累计打卡天数 |
| 连续天数 | 数字 | 连续打卡天数 |
| 创建日期 | 日期 | 创建日期 |
| 最后打卡日期 | 日期 | 最后打卡日期 |
| 是否启用 | 复选框 | 是否启用 |

#### 表4：打卡记录表

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 日期 | 日期 | 打卡日期 |
| 习惯名称 | 文本 | 关联习惯名称 |
| 复盘 | 文本 | 打卡复盘内容 |
| 是否打卡 | 复选框 | 是否已打卡 |

#### 表5：灵感表

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 灵感内容 | 文本 | 灵感内容 |
| 标签 | 多选 | 健康/学习/工作/生活/爱好 |
| 创建时间 | 日期时间 | 创建时间 |

### 1.2 创建飞书任务清单

1. 打开飞书 App → 工作台 → 任务
2. 创建三个清单：「学习」「工作」「生活」
3. 记录每个清单的 UUID（从清单 URL 获取）

## 2. 获取 Base Token

### 2.1 通过飞书开放平台获取

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用
3. 在「凭证与基础信息」中获取 `App ID` 和 `App Secret`
4. 调用接口获取 `user_access_token` 或 `tenant_access_token`

### 2.2 通过 lark-cli 获取

```bash
# 登录飞书
lark-cli auth login

# 获取 token 信息
lark-cli auth status
```

---

## 3. 获取 Table ID

### 3.1 通过多维表格 URL

打开你的多维表格，URL 格式如下：

```
https://xxx.feishu.cn/base/[BASE_TOKEN]?table=[TABLE_ID]
```

- `BASE_TOKEN`：多维表格的唯一标识
- `TABLE_ID`：每个数据表的唯一标识

### 3.2 通过 lark-cli 获取

```bash
# 列出所有表
lark-cli base +table-list --base-token "YOUR_BASE_TOKEN"
```

## 4. 配置脚本

### 4.1 下载脚本

将 `scripts/` 目录下的所有脚本下载到本地。

### 4.2 替换占位符

编辑每个脚本，替换以下占位符：

| 占位符 | 说明 | 获取方式 |
|--------|------|----------|
| `YOUR_BASE_TOKEN` | 多维表格 Token | 见上方「获取 Base Token」 |
| `YOUR_BASE_URL` | 飞书多维表格地址 | 如 `https://xxx.feishu.cn/base/YOUR_BASE_TOKEN` |
| `YOUR_SCHEDULE_TABLE_ID` | 日程表 ID | 在多维表格中打开「固定日程表」 |
| `YOUR_TASK_TABLE_ID` | 任务表 ID | 在多维表格中打开「任务表」 |
| `YOUR_HABIT_TABLE_ID` | 习惯表 ID | 在多维表格中打开「习惯表」 |
| `YOUR_CHECKIN_TABLE_ID` | 打卡记录表 ID | 在多维表格中打开「打卡记录表」 |
| `YOUR_IDEA_TABLE_ID` | 灵感表 ID | 在多维表格中打开「灵感表」 |
| `YOUR_STUDY_LIST_ID` | 学习清单 UUID | 飞书任务-学习清单 URL |
| `YOUR_WORK_LIST_ID` | 工作清单 UUID | 飞书任务-工作清单 URL |
| `YOUR_LIFE_LIST_ID` | 生活清单 UUID | 飞书任务-生活清单 URL |

### 4.3 脚本使用示例

```bash
# 给脚本添加执行权限
chmod +x scripts/*.sh

# 创建日程
./scripts/create_schedule.sh "每天早起" "07:00" "每天" "健康"

# 创建任务
./scripts/create_task.sh "完成报告" "2026-04-20" "工作" "高"

# 创建习惯
./scripts/create_habit.sh "阅读30分钟" "学习"

# 打卡
./scripts/checkin_habit.sh "阅读30分钟" "今天读了《xxx》，很有收获"

# 记录灵感
./scripts/record_idea.sh "想学习一门新技能"

# 查看今日汇总
./scripts/query_today.sh

# 同步飞书数据
./scripts/sync_from_feishu.sh
```

---

## 5. 注意事项

1. **时区设置**：脚本使用北京时间（Asia/Shanghai）
2. **重复事件**：飞书日历的重复事件需要在飞书网页端创建
3. **Token 有效期**：user_access_token 有有效期限制，注意及时刷新
4. **数据安全**：请勿将你的 Token 上传到公开仓库
