# 成长计划助手

> 一个基于扣子(Coze)的个人成长管理 Skill，通过自然语言交互管理日程、任务、习惯，支持飞书日历/任务双向同步。

## 项目背景

一个个人效率工具，用于管理日常工作学习中的日程、任务和习惯打卡。

**设计理念**：用自然语言代替复杂的 App 操作，一句话完成记录，定时自动回顾总结。

## 功能一览

| 模块 | 功能 | 飞书同步 |
|------|------|----------|
| 日程管理 | 创建/修改/删除重复日程 | 飞书日历 |
| 任务管理 | 创建/完成/复盘任务 | 飞书任务 |
| 习惯打卡 | 打卡、连续天数统计 | - |
| 灵感记录 | 随时记录想法 | - |
| 回顾总结 | 今日汇总、周回顾、月回顾 | - |

## 文件结构

```
growth-plan-assistant/
├── README.md               # 项目说明（本文件）
├── SETUP.md                # 配置指南
├── SKILL.md                # Skill 主文件（触发词、功能说明）
├── scripts/                # Shell 脚本
│   ├── create_task.sh      # 创建任务
│   ├── complete_task.sh    # 完成任务
│   ├── checkin_habit.sh    # 习惯打卡
│   ├── query_today.sh      # 今日汇总
│   └── ...                 # 更多脚本
└── references/
    └── database-config.md  # 飞书多维表格配置说明
```

## 技术栈

- **平台**：扣子 (Coze) - AI Agent 开发平台
- **数据存储**：飞书多维表格
- **同步**：飞书日历、飞书任务
- **脚本**：Shell + lark-cli (飞书命令行工具)

## 使用方式

在扣子平台安装此 Skill 后，通过触发词激活：

```
成长，添加日程：每天早上8点起床
成长，添加任务：投简历5个 截止4月20日 工作
成长，打卡：快走10000步
成长，灵感：学一下 AI 绘画
成长，今日汇总
成长，周回顾
```

## 快速开始

1. Fork 本仓库
2. 按照 [SETUP.md](./SETUP.md) 配置飞书多维表格
3. 在扣子平台创建 Bot，导入 Skill 文件夹
4. 开始使用
